# Miles-final-website
A personal website showcasing my journey as a Computer Technology senior at Bowie State University. Built to highlight my projects, resume, and professional links in a visually engaging way.
This includes 
- A welcome section introducing my background and story
- An about section with my skills and experiences
- Project links to GitHub repositories
- Resume section with PDF download and QR code
- Connection section with GitHub and LinkedIn links
- Integrated chatbot powered by Dialogflow Messenger
